# Интерфейсы

## Список интерфейсов

- ✅ [**MaxApiConfigInterface**](../src/Contracts/MaxApiConfigInterface.php) — An interface implementing
  the configuration storage.
- ✅ [**ModelInterface**](../src/Contracts/ModelInterface.php) — Interface for models with Raw data.

### Конвенции

- Формат элемента списка:
    ```markdown
    - ❌ [**InterfaceName**](../src/Contracts/InterfaceName.php) — Короткое описание интерфейса
    ```
- Добавляемые интерфейсы, нужно отмечать символом `❌`
- Список должен быть отсортирован по алфавиту
- **Отметка `✅` ставится при выполнении всех условий:**
    - Есть описание интерфейса на английском языке.
    - Все элементы интерфейса имеют комментарии на английском языке.
